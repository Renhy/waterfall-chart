# waterfall-chart
# 瀑布图控件

###介绍
在振动分析的应用场景中，需要将连续时刻的频域信号叠加显示，以观察振动信号的变化趋势。  
常见的图谱都是基于XY轴建立的笛卡尔坐标系，可以绘制(x, y)数据点的应用。  
而在振动分析中，每一组数据都是在某个具体时间点上的一组频域信号：
